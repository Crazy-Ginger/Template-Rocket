# Generic Rocket Files using OpenSCAD
A basic library that allows for quick and easy mockups of basic model rocketry components.

## Includes
---

### Nodes Cones:
- Conical
- Parabolic
- Elliptical
- Tangental
- Power Series
- Sears-Haack Series

### Fins:
- Basic flat trapezoid

### Other
- Tubes
- Centering Rings
